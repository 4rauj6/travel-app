Como trabalhar com formulários dentro do HTML?

Para se trabalhar com formulários dentro de um arquivo HTML é necessário mexer com as "Tags de formulários" elas são: "form e method". As duas Tags devem ir juntas sempre que se for trabalhar com formulários. O a tags "Form" define qual tipo de formlário vai ser tratado e enviado para qual lugar. Já a tag "Method" define qual ação será tomada pela pagina após a interação do usuário com o formulário. Podemos utilizar um formulário para diversas ações dentro da pagina HTML. Um dos usos mais comuns é em paginas de login e cadastros dentro de um sistema. Com a utilização do formulário podmeos coletar e armazenar os dados do usuário dentro do sistema.
